# Command Line on Terminal
---

## 📌 1. ระบุตำแหน่งปัจจุบัน
คำสั่งสำหรับตรวจสอบว่าตอนนี้เราอยู่ในโฟลเดอร์ไหนของระบบ
```bash
pwd
```
## 📂 2. List – คำสั่งสำหรับแสดงแฟ้มข้อมูล
> แสดง file / folder ที่อยู่ใน folder ปัจจุบัน
```bash
ls
```
> 📝 แสดงรายละเอียดไฟล์แบบยาว (Permission / จำนวน user / ขนาดไฟล์ / วันเวลา / ชื่อไฟล์)
```bash
ls -l
```
> 📏 แสดงรายละเอียดไฟล์แบบอ่านง่าย (human-readable) เช่น ขนาดไฟล์เป็น M (Megabyte)
```bash
ls -lh
```
> ⏱️ เรียงไฟล์ตามเวลาที่แก้ไขล่าสุด โดยไฟล์ใหม่ที่สุดจะแสดงอยู่ด้านล่างสุด -t → เรียงตามเวลา -r → กลับลำดับ
```bash
ls -ltr
```
**การอ่าน Permission เบื้องต้น**
* 🔍 r → read (อ่าน)
* ✏️ w → write (เขียนหรือแก้ไข)
* ⚙️ x → execute (เรียกใช้งาน)

💡 สามารถกดลูกศรชี้ขึ้น [Up Arrow] เพื่อไปที่คำสั่งก่อนหน้าที่เราเคยใช้

### 📁 3. Change Directory (เปลี่ยนตำแหน่งโฟลเดอร์)
* ถอยออกมาสุด กลับไปยัง Home directory (~)
```bash
cd
```
* ย้อนกลับไปโฟลเดอร์ปัจจุบัน 1 ระดับ
```bash
cd ..
```
* ย้อนกลับ 2 ระดับ
```bash
cd ../..
```
* เข้าไปยังโฟลเดอร์ที่ต้องการ (สามารถกด Tab เพื่อช่วยเติมชื่อโฟลเดอร์อัตโนมัติ)
```bash
cd <folder_name>/
```
### 🗂️ 4. File / Folder Management (การจัดการไฟล์และโฟลเดอร์)
**สร้างโฟลเดอร์ (Make Directory)**
```bash
mkdir <folder_name>
```
**สร้างและแก้ไขไฟล์ด้วย vi**
```bash
vi <file_name>
vi <file_name>.py
```
**🛠️ ขั้นตอนการใช้งาน vi**
* กด i → เพื่อเข้าสู่โหมดเขียนหรือแก้ไข
* กด Esc → เมื่อพิมพ์เสร็จ
* พิมพ์ :wq → เพื่อบันทึกและออก
* หรือพิมพ์ :q! → เพื่อออกโดยไม่บันทึก

**เปิดดูเนื้อหาไฟล์ (Open File)**
```bash
cat <file_name>
```
**ลบไฟล์และโฟลเดอร์ (Delete)**
* ลบไฟล์
```bash
rm <file_name>
```
* ลบโฟลเดอร์ (ใช้ -r เพื่อให้มัน recursive ลบทุกอย่างข้างใน เพื่อลบทั้งโฟลเดอร์)
```bash
rm -r <folder_name>
```
### 🚚 5. Move File (ย้ายไฟล์ หรือ เปลี่ยนชื่อ)
**วิธีย้ายไฟล์แบบที่ 1** ระบุ path ต้นทางและปลายทาง
```bash
mv <ที่อยู่ต้นทาง>/<ชื่อไฟล์> <ที่อยู่ปลายทาง>/.
```
**วิธีย้ายไฟล์แบบที่ 2** เข้าไปอยู่ในโฟลเดอร์ปลายทางก่อน
```bash
mv ../<ชื่อไฟล์> .
```
> (.) หมายถึงตำแหน่งปัจจุบันที่เราอยู่

**วิธีเปลี่ยนชื่อไฟล์ (Rename)**
```bash
mv <ชื่อเก่า> <ชื่อใหม่>
```

### 📄 6. Copy File / Folder (คัดลอกไฟล์และโฟลเดอร์)
**Copy ไฟล์**
```bash
cp <ที่อยู่ต้นทาง>/<ชื่อไฟล์> <ที่อยู่ปลายทาง>/.
```
**Copy โฟลเดอร์ (ต้องใส่ -r เสมอ)**
```bash
cp -r <ที่อยู่โฟลเดอร์ต้นทาง> .
```
### 📖 7. Manual Page (หน้าคู่มือคำสั่ง)
```bash
man <command>
```
> ใช้ดูคู่มือคำสั่งและโปรแกรมต่างๆ เช่น man ls

### 🖥️ 8. Check System Preference (ตรวจสอบระบบ)
```bash
htop
```
> 📊 ใช้ดูการใช้งานของเครื่อง เช่น RAM และ CPU (ต้องทำการ Install ก่อนใช้งาน)
